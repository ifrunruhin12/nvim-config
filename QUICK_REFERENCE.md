# Quick Reference Guide

## Terminal (NEW!)

### Quick Access
- **`Ctrl + \`** - Toggle terminal at bottom (quickest way!)
- **`<leader>tt`** - Toggle horizontal terminal
- **`<leader>tf`** - Toggle floating terminal
- **`<leader>tv`** - Toggle vertical terminal

### Inside Terminal
- **`Esc`** - Exit to normal mode
- **`Ctrl + w` then `+/-`** - Resize height
- **`Ctrl + w` then `>/<`** - Resize width

## File & Code Search

### Find Files
- **`<leader>ff`** - Find files (includes hidden files)
- **`<leader>fr`** - Recent files

### Search in Code (Codebase-wide)
- **`<leader>fg`** - Live grep - search for any keyword across ALL files
- **`<leader>fw`** - Find word under cursor in all files

### Other
- **`<leader>fb`** - Switch between open buffers
- **`<leader>fc`** - Find commands

## Navigation in Telescope
- **`Ctrl + j/k`** - Move up/down in results
- **`Enter`** - Open file
- **`Ctrl + x`** - Open in horizontal split
- **`Ctrl + v`** - Open in vertical split
- **`Esc`** - Close

## Neo-tree File Explorer
- **`<leader>e`** - Toggle neo-tree
- **`<leader>o`** - Focus neo-tree
- **`H`** (when in neo-tree) - Toggle hidden files visibility
- **`?`** (when in neo-tree) - Show all keybindings

### Inside Neo-tree
- **`Enter`** - Open file
- **`s`** - Open in vertical split
- **`S`** - Open in horizontal split
- **`a`** - Add file
- **`A`** - Add directory
- **`d`** - Delete
- **`r`** - Rename

## Basic Vim
- **`<leader>w`** - Save file
- **`<leader>q`** - Quit

---

**Note**: `<leader>` is the Space key in your config
